# CMS-JSP

##### Über das Projekt:

Hier geht es um einem Webapp, das auf tomcat-Server läuft. um es zu auszuprobieren, laden zie die War-Datei herunter und stationieren Sie diese auf Ihren Apache-tomcat-Server. Wie der Titel schon sagt, es ist ein Content Manage System(CMS) auf deutsch auch Inhaltverwaltungssystem für die Darstellung einer Webseite mit sehr einfach Strucktur. hier weiter zur Documnetation
